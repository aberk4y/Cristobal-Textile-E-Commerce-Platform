## Project Overview

Cristobal is a full-stack textile e-commerce platform developed using Node.js and Express. The application simulates a real-world online retail system where users can browse products, manage shopping carts, authenticate accounts, and interact with dynamic content.

The system follows a modular backend architecture with structured routing, middleware logic, database integration, and localization support.

## Installation

1. Clone the repository:

git clone https://github.com/aberk4y/Tekstil-Alisveris-Web-Sitesi.git

2. Install dependencies:

npm install

3. Start the server:

node server.js

4. Open in browser:

http://localhost:3000
